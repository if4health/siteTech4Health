Sobre

O grupo de pesquisa IF4Health foi fundado em 2020 e é composto atualmente por 3 professores doutores do campus Charqueadas, nas áreas de Engenharia Elétrica e Ciência da Computação. O objetivo do grupo é realizar pesquisas aplicadas na área biomédica, focando a conectivdade dos sistemas, de forma interdisciplinar.

Projetos

BIoT - Endpoint IoT para aquisição de biopotenciais compatível com protocolos de interoperabilidade na saúde

Em Andamento Projeto de pesquisa aprovado no Edital PROPESP 09-2022: O objetivo do projeto é desenvolver uma placa eletrônica de aquisição de biopotenciais com segurança e confiabilidade e um sistema embarcado capaz de processar e enviar os biopotenciais ao servidor através da internet. CloudECG - Interoperabilidade e Segurança para Registros Eletrônicos de Saúde de Eletrocardiogramas (ECG) na Nuvem

Em Andamento Projeto de pesquisa aprovado no Edital PROPESP 09-2022: O objetivo geral deste trabalho é utilizar os padrões internacionais FHIR e openEHR para garantir interoperabilidade, segurança e privacidade de dados na transmissão e na autenticação de exames ECG em dispositivos ECG e em aplicações web por meio da API CloudECG. Em Andamento Projeto de pesquisa aprovado no Edital PROPESP 09-2022: Esse projeto tem por objetivo utilizar os sinais de ECG adquiridos pelo hardware desenvolvido dentro do grupo para classificar arritmias de forma inteligente. Serão desenvolvidas técnicas para classificação embarcada e na nuvem.

Internet das Coisas Aplicado ao Desenvolvimento de Sistemas para Diagnóstico Remoto de Cardiopatias

Finalizado Projeto de Inovação aprovado no Edital PROPESP 06-2020: No exemplo da figura, na linha de pesquisa IoT in HealthCare, teve alunos da ECA desenvolvendo dispositivo físico para aquisição de sinais de eletrocardiograma (ECG device). O sistema embarcado serve para exibir o ECG localmente e enviar os dados para internet. Também houve alunos do TSI no back-end para armazenamento e processamento do ECG e no front-end para visualização do ECG. O projeto inteiro foi executado durante a pandemia, impondo enormes dificuldades de toda ordem: comunicação, infraestrutura, mental, etc...
